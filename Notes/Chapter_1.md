# Lesson 1
## Setting Up Your Dev Environment
1. Go To [VS Code Download Page](https://code.visualstudio.com/download)
2. Download The Revision for your OS
3. Go To [VS Code C/C++ Development](https://code.visualstudio.com/docs/languages/cpp)
4. Test Development Environment with Hello World Program

## Set Up Github
1. [Git Hub](http://github.com)
2. Set Up Github Account
3. [Git Hub VS Code](https://code.visualstudio.com/docs/sourcecontrol/github)
4. Create your repository in Github
5. Retrieve Your Repository
    1. Type CMD+SHIFT+P "Git Clone"
    2. Sign In to your account
    3. Select Your Created Repository
