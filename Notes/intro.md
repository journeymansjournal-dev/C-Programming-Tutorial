# C Programming Tutorial Introduction

## Why Choose C

1. A general purpose programming language
    1. Can be used for software in various domains
    2. One of the foundational high-level GLP languages
    3. While C is useful and commonly used in Operating Systems and Compilers it is not specific to this domain.

2. Support for Low Level Functions
    1. Exposes you to the computers actual function

3. Simple and Small Language things not just programs)    


## What are C's Features
1. Type Support
    1. Primitive
        1. Characters 
        2. Intergers
        3. Floating Point Numbers

    2. Derived Data Types
        1. Pointers
        2. Arrays
        3. Structures
        4. Unions

3. Control Flow
    1. Conditional Statements (if-else) (switch)
    2. Looping (while) (for)

4. Functions 
    1. May return values of primitive or derived types
    2. Internal Variable and External Variables

5. I/O Support
    1. Read Files
    2. Write Files

6. Macro Support
